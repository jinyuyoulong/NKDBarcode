– initiator

Returns the encoded initiator or alignment bar(s)

- (NSString *)initiator

Discussion

@method initiator

This is likely the most complex initator method in the framework. Because there are three code sets to choose from, we have to determine which one is the best choice for our content. Our algorithm will be to check first to see if there are only numbers, then we use set C. If there are other characters, we will sum the number of lower-case letters and the number of non-printing ASCII characters. If there are more lowercase letters than non-printings, we will use set B, otherwise, we will use set C.
