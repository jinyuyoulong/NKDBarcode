Overview

@category NSImage(Normalize)

This category adds two instance methods that operate on the resolution of an NSImage with at least one NSBitmapImageRep

– normalizeSize

Normalize the resolution to 72 dpi in lossless fashion so that the image displays at full size onscreen.

- (UIImage *)normalizeSize

Return Value

NSImage with the exact same data as the instance it is called on except that the resolution pixelHeight will be the same as the height and the pixelWidth will be the same as the width.

– setDPI:

Allows you to adjust (in a lossless fashion) the resolution of the images NSBitmapImageRep (if it has one) to a given number of points per inch.

- (UIImage *)setDPI:(int)dpi

Parameters

dpi

Integer value representing the desired horizontal and vertical pixels per inch.

Return Value

Self with the data unaltered except for the height and width (pixelHeight and pixelWidth remain unchanged) of the first NSBitmapRep the image contains.
