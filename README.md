ARC 版
这是一个日本人写的生成条形码的库, 写得还不错, 各种常用码都可以生成, 而且调用起来也很方便, 
唯一不足的就是这个库是MRC的, 所以我改成ARC了，供大家使用。cocoapods已经集成，可以使用
